Projects

  • TripPlanner

    The TripPlanner web application is written in JavaScript. It helps users find and display restaurants and attractions in a selected city. Using the Google Maps API, Travel Advisor API, and Google Places API, it provides detailed information and search features. The app includes a chat functionality for discussing and adding to the itinerary. TripPlanner's back-end is powered by Flask with MongoDB for data storage. It aims to offer a convenient platform for exploring dining and sightseeing options.

  • CurrencyExchanger

    The CurrencyExchanger mobile application is written in Kotlin. The app is designed to provide users with real-time currency conversion rates and exchange services. This application leverages Retrofit to fetch the latest currency data from the ExchangeRate-API. Currency Exchanger utilizes the Jetpack Compose library for a modern and intuitive user interface, ensuring a seamless and engaging experience. The app aims to simplify currency conversions for travelers, business professionals, and anyone needing accurate exchange rates.

  • PrayerWay (in development)

    The PrayerWay App is an iOS app written in Swift designed to assist Muslims in conveniently locating nearby mosques and determining their ability to attend congregational prayers on Fridays. This app gathers users' locations and employs a web scraper to identify mosques in their vicinity, ensuring they are well-informed about their options for prayer locations and times. The app then calculates if the user can make the congregation based on travel time.

  • TurboTrivia (in development)

    The TurboTrivia web application is written in JavaScript. The website is designed to quiz users on the history of cars. This application uses MongoDB to cycle through the database for questions for the user. TurboTrivia utilizes the React library as the front-end user interface for a smooth experience. TurboTrivia aims to provide an immersive experience for automotive enthusiasts and others who may just simply enjoy trivia.
